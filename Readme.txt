This script will export an XLSX file with 3 Worksheets:
Retention and label policies, Label Information and Policies by SharePoint Site

Retention and Label policies will list all Retention Policies and Label Policies in your tenant and the labels within them as well as the locations that they are applied (SharePoint Sites)

Label Information will provide information on the labels available to the tenancy including the retention action, duration and trigger

Policies by SharePoint site will list all policies applied to each SharePoint site and is intended to be searchable by site.
